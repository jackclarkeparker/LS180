INNER JOIN

Write an SQL query to display all devices along with all the parts that make them up.
We only want to display the name of the devices.
Its parts should only display the part_number.

Expected output:

     name      | part_number
---------------+-------------
 Accelerometer |          12
 Accelerometer |          14
 Accelerometer |          16
 Gyroscope     |          31
 Gyroscope     |          33
 Gyroscope     |          35
 Gyroscope     |          37
 Gyroscope     |          39
(8 rows)



SELECT name, part_number FROM devices
    INNER JOIN parts
      ON parts.device_id = devices.id;
